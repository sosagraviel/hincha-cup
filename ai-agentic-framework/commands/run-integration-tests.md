# Run Integration Tests

Executes all initialize-project agent integration tests.

## Instructions

Simply run: `bash ai-agentic-framework/tests/run-tests.sh`

This script will:
1. Execute each agent (01-04) against each test project
2. Validate JSON output from each agent
3. Report results

## Expected Output

The script shows:
- Total tests: X
- ✓ Passed: Y
- ✗ Failed: Z
- Final status
