# Integration Skills

Skills for integrating with external services and platforms like Jira, Confluence, GitHub, Notion, and Figma.

## Skills in this group

- **fetch-ticket-context**: Fetch complete context for a Jira ticket including external documentation from Notion, Confluence, and linked resources
- **figma-design-fetcher**: Fetch Figma designs, export frame images at 2x, and extract structured design constraints (layout, colors, typography, dimensions)
- **jira**: Manage Jira issues, projects, and workflows via the Atlassian MCP (create, update, transition, search with JQL)
- **mastering-confluence**: Comprehensive Confluence documentation management — pages, CQL search, ADF/markdown conversion, Git-to-Confluence sync
- **mastering-github-cli**: GitHub Actions and `gh` CLI operations — workflow logs, PRs, issues, artifacts, workflow triggers
- **notion-document-manager**: Manage Notion documents with smart chunking for large pages, search, and batch operations
