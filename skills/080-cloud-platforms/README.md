# Cloud Platforms Skills

Skills for working with cloud platforms and their CLIs/SDKs.

## Skills in this group

- **mastering-aws-cdk-plugin**: AWS Cloud Development Kit (CDK)
- **mastering-aws-cli**: AWS CLI v2 commands and workflows
- **mastering-gcloud-commands**: Google Cloud Platform (gcloud) CLI
- **using-firebase**: Firebase development for GCP-hosted applications
