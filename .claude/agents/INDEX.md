## Usage

Agents are specialized subprocesses that can be invoked for specific tasks:

```bash
# Planning phase
/Task with planner agent for implementation strategy

# Implementation phase
/Task with implementer-typescript for feature development

# Testing phase
/Task with tester-unit-typescript for test coverage
```

Each agent is configured with:
- Project-specific tech stack knowledge
- Common commands and workflows
- Implementation guidelines
- Testing patterns
